# 实验四成果目录

本目录对应《实验内容四.docx》中的要求：了解 XP 开发方法、了解 DevOps、完成活动图练习和项目活动分解，找出关键路径，并将工作文档保存到协作开发平台。

## 文件清单

| 文件 | 说明 |
|---|---|
| `experiment_04_report.md` | 实验四主报告，包含 XP、DevOps、活动图方法、项目活动分解和关键路径结论。 |
| `activity_network.md` | 人才招聘系统项目活动网络、最早/最晚时间、时差和关键路径分析。 |
| `devops_plan.md` | 结合当前 Qt 项目的轻量 DevOps 实施方案。 |

## 审阅重点

1. XP 与 DevOps 的理解是否贴合课程要求。
2. 活动分解和关键路径是否适合当前人才招聘系统。
3. 活动图练习、项目活动分解和关键路径分析是否完整。
