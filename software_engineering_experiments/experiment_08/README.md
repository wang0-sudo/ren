# 实验八成果目录

本目录对应《实验内容八.docx》中的要求：学习 Petri 网基础知识，使用动态建模工具补充人才招聘系统 SRS，对系统运行过程、状态变化、并发同步关系和关键业务约束进行建模，并更新项目跟踪表。

## 文件清单

| 文件 | 说明 |
|---|---|
| `experiment_08_report.md` | 实验八主报告，包含实验目的、Petri 网学习总结、动态建模过程、SRS 补充和项目跟踪更新。 |
| `dynamic_models.md` | 系统动态模型，包含状态图、活动图、数据流图和 OCL/逻辑约束。 |
| `petri_net_model.md` | 基于 Petri 网的人才招聘系统业务流程建模与性质分析。 |
| `srs_updates.md` | 对实验七 SRS 初稿的补充条目、约束说明和项目跟踪表。 |

## 审阅重点

1. 动态模型是否覆盖注册审核、岗位发布、岗位申请、留言处理等主要业务流程。
2. Petri 网模型是否清楚体现并发、同步、冲突和状态迁移关系。
3. SRS 补充是否能直接并入后续需求规格说明书定稿。
