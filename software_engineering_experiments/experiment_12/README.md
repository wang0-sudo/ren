# 实验十二成果目录

本目录对应《实验内容十二.docx》中的要求：结合项目开发历程评估模块设计，学习依赖注入技术，论述面向对象设计原则，并更新项目跟踪表。

## 文件清单

| 文件 | 说明 |
|---|---|
| `experiment_12_report.md` | 实验十二主报告，说明实验目的、设计原则学习、项目评估和结论。 |
| `design_principles_analysis.md` | 结合人才招聘系统说明六项面向对象设计原则的应用。 |
| `dependency_injection_notes.md` | 依赖注入学习笔记，以及当前项目中的应用和改进方案。 |
| `module_design_evaluation.md` | 对 `LoginWindow`、`DashboardWindow`、`RecruitmentService` 等模块的设计评估。 |
| `refactoring_recommendations.md` | 面向设计原则的重构建议和分阶段实施方案。 |
| `project_tracking.md` | 实验十二项目进度、工作量、里程碑和风险跟踪表。 |

## 审阅重点

1. 设计原则是否结合当前项目代码进行说明，而不是只停留在概念解释。
2. 依赖注入分析是否能对应 `std::shared_ptr<RecruitmentService>` 的实际用法。
3. 改进建议是否符合课程项目规模和当前架构。
