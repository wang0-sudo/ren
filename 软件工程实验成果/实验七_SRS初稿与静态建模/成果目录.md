# 实验七成果目录

本目录对应《实验内容七.docx》中的要求：写作项目 SRS 初稿，学习 SRS 文档要求和特点，并使用 E-R、UML 等静态建模工具对项目建模。

## 文件清单

| 文件 | 说明 |
|---|---|
| `实验七报告.md` | 实验七主报告，包含 SRS 案例调研、模板对比、静态建模和项目跟踪更新。 |
| `SRS初稿.md` | 人才招聘系统 SRS 初稿。 |
| `静态模型.md` | 用例模型、领域类图、ER 模型和包/模块结构图。 |
| `SRS模板对比.md` | 国标 SRS 模板与《掌握需求过程》需求模板的对比和取舍。 |

## 审阅重点

1. `SRS初稿.md` 是否可作为后续 SRS 定稿基础。
2. 静态模型是否覆盖个人用户、企业用户、管理员、岗位、简历、留言等核心对象。
3. 模板对比中选取的 SRS 章节是否适合课程项目。
