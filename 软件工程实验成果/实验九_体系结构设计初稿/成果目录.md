# 实验九成果目录

本目录对应《实验内容九.docx》中的要求：进一步学习 UML 和计算机逻辑，参考 GB 软件设计说明与 ISO/IEC/IEEE 42010 软件体系结构描述思想，完成项目 SAD 初稿，补充经典体系结构案例研究，并形成 SRS 定稿和项目跟踪更新。

## 文件清单

| 文件 | 说明 |
|---|---|
| `实验九报告.md` | 实验九主报告，说明实验目标、完成过程、产出和结论。 |
| `SAD初稿.md` | 人才招聘系统软件体系结构设计文档 SAD 初稿。 |
| `架构视图.md` | 架构视图集合，包含上下文视图、逻辑视图、组件视图、运行视图、部署视图和数据视图。 |
| `标准对比与案例研究.md` | GB 软件设计说明与 ISO/IEC/IEEE 42010 对比，以及经典架构案例对本项目的启示。 |
| `SRS定稿.md` | 在实验七 SRS 初稿和实验八动态模型基础上整理的 SRS 定稿。 |
| `项目跟踪表.md` | 本周项目进度、里程碑和工作量跟踪表。 |

## 审阅重点

1. SAD 是否能支撑后续详细设计和测试设计。
2. 架构视图是否与当前 `Qt图形界面` 代码结构一致。
3. SRS 定稿是否整合了静态模型、动态模型和业务约束。
