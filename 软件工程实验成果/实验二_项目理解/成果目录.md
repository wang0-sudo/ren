# 实验二成果目录

本目录对应《实验内容二.docx》中的要求：持续项目沟通调查、修订可行性分析报告、调研并选用 CASE 工具，并将小组工作文档和进度保存到协作开发平台。

## 文件清单

| 文件 | 说明 |
|---|---|
| `实验二报告.md` | 实验二主报告，包含可行性分析报告修订版、CASE 工具调研、工具选型和初步应用成果。 |
| `项目跟踪表.md` | 项目跟踪表，记录阶段任务、负责人角色、计划周期、状态和产出。 |

## 审阅重点

1. 项目范围是否仍以“人才招聘系统”为准。
2. CASE 工具组合是否采用 `StarUML + PlantUML/Mermaid + GanttProject + GitHub Projects + Git`。
3. 可行性分析中的风险、进度和技术路线是否需要按实际课程安排调整。
