# 实验五成果目录

本目录对应《实验内容五.docx》中的要求：深入理解活动图，练习项目跟踪工具，讨论人力资源管理和软件开发团队组织结构。

## 文件清单

| 文件 | 说明 |
|---|---|
| `实验五报告.md` | 实验五主报告，包含活动图时间分析、项目跟踪工具、人力资源组织结构和工作方式分析。 |
| `活动时间分析.md` | 根据实验五活动图计算事件最早/最晚时间、活动最早/最晚开始时间、时差和关键路径。 |
| `项目跟踪甘特图.md` | 人才招聘系统项目甘特图、任务跟踪表和每周更新记录。 |
| `团队组织.md` | 国内外软件开发团队组织结构、工作方式对比和团队管理建议。 |

## 审阅重点

1. 活动图计算中关键路径是否接受为 `A -> D -> H -> L -> N -> O -> P`。
2. 项目跟踪表是否适合后续每周更新。
3. 团队组织结构和工作方式分析是否符合课程报告口径。
