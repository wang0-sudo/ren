# 实验十一成果目录

本目录对应《实验内容十一.docx》中的要求：继续完成体系结构设计任务，完成自己项目的 SAD，并更新项目跟踪表。

## 文件清单

| 文件 | 说明 |
|---|---|
| `实验十一报告.md` | 实验十一主报告，说明 SAD 完成过程、评估方法和结论。 |
| `SAD定稿.md` | 人才招聘系统软件体系结构设计文档 SAD 定稿。 |
| `架构评估.md` | 架构评估，包含质量属性场景、风险点、权衡分析和改进建议。 |
| `架构改进计划.md` | 架构改进计划，说明短期、中期和长期演进路线。 |
| `SAD评审清单.md` | SAD 提交前评审清单。 |
| `项目跟踪表.md` | 实验十一项目进度、工作量和风险跟踪表。 |

## 审阅重点

1. `SAD定稿.md` 是否可以作为提交给助教的体系结构设计文档。
2. 架构评估是否覆盖可维护性、可靠性、安全性、可测试性和可扩展性。
3. 改进计划是否与当前 Qt 桌面端项目实际情况匹配。
